Como se alistar no Exército?

Aqui são os passos para se alistar:

  1. Vá para o link
  2. Entre em contato com um ADM
  3. Faz o seu cadastro
  4. Ao terminar esperar 1 semana
  5. Se você recebeu um Email, parabéns você passou.

Agora regras dentro do quartel

  • Não é permitido nenhum tipo de contato com familiares ou parentes.
  • Todas as ordem dadas por um Sargento+ devem se seguindas.
  • Os recrutas/cadetes não estão autorizados a usar ou manusear Armas de fogo.
  • Numca deve atrapalha os serviços das Forças Especiais.
  • Se você ver algum membro quebrado essas regras, reporte para um Tenente+
